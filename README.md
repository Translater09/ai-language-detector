# AI Language Detector & Translator

This project is a web application that detects the language of a given text and translates it into a selected target language.

## Features
- Automatic language detection
- Confidence score for detection
- Multi-language translation (EN, TR, ES, FR, DE, IT)
- Flask backend + Vanilla JS frontend


## Technologies
- Python (Flask)
- HTML / CSS / JavaScript
- Language Detection API
- Translation API
